\[\Ack(m, n) = \left\{\begin{array}{ll}
n + 1, & m = 0 ,\\
\Ack(m - 1, 1), & n = 0 ,\\
\Ack(m-1, \Ack(m, n-1)), & m > 0 \text{\,且\,} n > 0 .\\
\end{array}\right.\]

使用递归可以实现：
\lstinputlisting[language=C,caption={\tt t5.c}]{../ex1/t5.c}

运行结果：
\begin{verbatim}

请输入 n: 3
Ackerman(2, 3) = 9
\end{verbatim}
